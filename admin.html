📦 Publish from GitHub Folder

Pick-and-publish. Upload your blog folder (HTML + images + PDFs + Excel) to github.com/vkstecho/VksTech. The dropdown below lists all blog folders on your GitHub repo — pick one, click Fetch & Preview, then Publish. Edit on GitHub anytime; the site updates automatically.

Your HTML file needs these lines in the <head>: 
<meta name="category" content="packaging">
<meta name="subcategory" content="metalliser">  <!-- optional, only for category=packaging -->
Category values: packaging · excel · career · tech · manufacturing. Sub-category values (only when category=packaging): metalliser · slitter · quality · general. Title comes from the first <h1>/<h2>, summary from the first <p>, cover from the first <img>. Bilingual: English from .en-content + Hindi from .hindi-section, joined with |.
Folders are scanned from github.com/vkstecho/VksTech. Only folders containing a .html file are shown. Already-published folders are hidden by default — tick the checkbox to see all.
Can't find your folder? Type it manually
Useful if GitHub's API rate limit was hit, or for sub-paths like blogs/my-folder.
