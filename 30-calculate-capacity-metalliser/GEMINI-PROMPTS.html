Review Checklist — 8-Point Check Before Publishing

  1. Brand colour discipline — only Navy / Orange / Teal / Cream / white / grey. No blue, no green outside teal, no yellow, no red other than the orange.
  2. Typography quality — clean sans-serif (Inter / Söhne / Helvetica), big numbers in heavy condensed font (Bebas / Oswald / Anton).
  3. Number accuracy (HARD) — Cycle anatomy: 67.5 / 20 / 20 / 10 / 0.9 → 118.4 min total → 12 rolls/day. Capacity contrast: 32 MT/day vs 12.1 MT/day, delta −17.6 = −62%. Roll weights: 2,683 kg vs 1,407 kg.
  4. Material name discipline — must say AlOx (not "Alubond"). PET density = 1.38 g/cc (matches Blog 27).
  5. Hero number sizing parity — for Image 3A/3B: both DAILY CAPACITY numbers (32 and 12.1 MT/day) MUST be the same point size. Different sizes misrepresent the comparison.
  6. Logo placement — bottom-right, ~80–110 px, on white backplate.
  7. Information density vs scannability — cycle anatomy reads as "5 components stack to 118 min" in 5 seconds; contrast image reads as "32 vs 14" in 3 seconds.
  8. No AI-tells — no distorted hands, garbled small text, AI watermarks, or unnatural plastic surfaces.

Troubleshooting

  • Numbers come out wrong: Always overlay correct numbers in an editor afterward. AI image models are unreliable with numerics.
  • × symbol or − sign renders as box: Add to prompt: Use Unicode multiplication × (U+00D7) and minus − (U+2212). If unable to render, use 'x' and '-' as substitutes.
  • Hindi renders as gibberish: Use Plan B — generate EN first, text-replace in Photopea/Canva using a real Devanagari font.
  • Decorative clutter (sparkles, hearts, confetti): Add: STRICTLY no decorative elements, no clipart, no flourishes. WSJ / Bloomberg / HBR editorial style ONLY.
  • Logo gets ignored: Always have Plan B ready — composite manually.
