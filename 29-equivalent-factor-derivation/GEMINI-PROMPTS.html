Review Checklist — 8-Point Check Before Publishing

  1. Brand colour discipline — only Navy / Orange / Teal / Cream / white / grey. No blue, no green outside teal, no yellow, no red other than the orange.
  2. Typography quality — clean sans-serif (Inter / Söhne / Helvetica), formulas in heavy condensed font (Bebas / Oswald / Anton).
  3. Formula accuracy (HARD) — every formula must be syntactically correct: R_base ÷ R_OD, Target Width ÷ Actual Width, etc. Worked numbers: 1.25× / 1.875× / 3.00× / 2.00× / 2.00× / ~11.4×.
  4. Material name discipline — must say AlOx (not "Alubond"), Reverse/Release Coat or RC, material codes in PET_12_M2.2 format.
  5. Productivity-vs-Geometry color discipline — Lane 1 (OD/Micron/Material) orange-themed, Lane 2 (Width/Length) teal-themed, Lane 3 (Combined) navy.
  6. Logo placement — bottom-right, ~80–110 px, on white backplate.
  7. Information density vs scannability — 5 boxes readable in 5 seconds; Combined formula is the visual punch line.
  8. No AI-tells — no distorted hands, garbled small text, AI watermarks, or unnatural plastic surfaces.

Troubleshooting

  • Formulas come out garbled / mathematically wrong: Always overlay correct formulas in an image editor afterward. Treat generated formulas as placeholders, not truth.
  • × symbol renders as box: Add to prompt: Use Unicode multiplication symbol × (U+00D7). If unable to render, use 'x' as substitute.
  • Numbers come out wrong: Always overlay correct numbers in an image editor.
  • Hindi renders as gibberish: Use Plan B — generate EN first, text-replace in Photopea/Canva using a real Devanagari font.
  • Decorative clutter (sparkles, hearts, confetti): Add: STRICTLY no decorative elements, no clipart, no flourishes. WSJ / Bloomberg / HBR editorial style ONLY.
  • Logo gets ignored: Always have Plan B ready — composite manually.
